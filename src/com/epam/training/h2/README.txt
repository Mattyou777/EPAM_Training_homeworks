Данная программа является демонстрацией навыков работы с БД H2. Для работы необходима среда H2 и БД test.
Если в указанной БД нет таблицы books или она пуста, то срабатывает заглушка автозаполнения, для демонстрации функционала приложения.
Приложение умеет добавлять книги по ISBN, выводить информацию о конкретной книге по ISBN, выводить список книг по диапазону ISBN, вывод отсортированных списков книг( по ISBN, дате добаления, автору и названию).
