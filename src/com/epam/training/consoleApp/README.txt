Информация по командам доступна по команде help.
В данный момент:
Реализованы методы копирования, удаления, переименования, архивирования, разархивирования, получения списка файлов и папок указанной директории.
Архивирование и разархивирование реализовано только для ZIP.
Разархивирование вложенных директорий работает некорректно.
Архивирование вложенных файлов происходит без разделения на папки.
